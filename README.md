# ☕ DAILY SPOT CAFE WEB SYSTEM

> A modern, responsive cafe website showcasing menu, products, and user-friendly navigation.

---

## 🛠 Tech Stack

**Frontend**
- HTML5
- CSS3
- JavaScript
- Bootstrap *(Framework)*

**Backend**
- PHP

**Database**
- MySQL

**Server**
- Apache

**Tools**
- VS Code *(Code Editor)*
- MySQL Workbench *(Database)*
- XAMPP *(Server)*
- Git & Github *(Version Control)*
---

## ⚙️ Installation

```bash
git clone https://github.com/CodeMico/daily-spot-cafe.git
cd daily-spot-cafe
```

---

## ▶️ How to Open the Project

1. Open XAMPP:
   - Click Start on Apache

2. Open in browser:

```
localhost/daily-spot-cafe
```
